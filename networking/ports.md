Networking - Ports and IP Addresses

ip addr
- ip = Linux networking tool
- addr = Address
- Shows the network interfaces and IP addresses of the computer.

Network Interface
- A connection that allows a computer to communicate on a network.

Loopback Interface (lo)
- Uses IP address 127.0.0.1
- Also called localhost.
- Allows a computer to communicate with itself.

eth0
- Main network interface in WSL.
- My IP address: 172.18.45.6

IP Address
- Identifies a computer on a network.

Port
- Identifies a specific service running on a computer.

Examples:
22 = SSH
80 = HTTP
443 = HTTPS
53 = DNS
