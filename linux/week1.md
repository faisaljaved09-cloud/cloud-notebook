GitHub	Website that stores Git repositories online
1. The user enters a website address (e.g., www.google.com) into a browser.

2. DNS converts the domain name into an IP address.

3. The browser sends a request to the server using the IP address.

4. The server processes the request and sends back HTML, CSS, JavaScript, images, and other resources.

5. The browser reads these files and renders them into the webpage displayed on the screen.
