
EC2 is a virtual computer/server provided by AWS.

Companies use EC2 because:
- No need to buy physical servers.
- Easy to scale resources.
- AWS manages the hardware.

EC2 provides:
- CPU
- RAM
- Storage
- Operating System
- IP Address

To connect:
SSH
Port 22

Example:
Laptop → SSH → EC2 Linux Server

Web traffic:
HTTPS
Port 443

Nginx:
A web server software that can run on an EC2 instance.
